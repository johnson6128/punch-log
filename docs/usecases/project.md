# ユースケース: プロジェクト管理

## UC-PJ01: プロジェクトの登録

- **アクター**: 管理者
- **概要**: 新しいプロジェクトをシステムに登録する。
- **基本フロー**:
  1. 管理者がプロジェクト名・期間・担当社員を入力する。
  2. システムがプロジェクトを保存し、工数・経費の紐付けが可能になる。
- **例外フロー**:
  - 同一名称のプロジェクトが既に存在する場合、警告を表示する（重複登録は許容するが注意を促す）。

---

## UC-PJ02: プロジェクト情報の更新・終了

- **アクター**: 管理者
- **概要**: プロジェクトの名称・期間・担当社員などを変更する。また、完了したプロジェクトをクローズする。
- **基本フロー**:
  1. 管理者がプロジェクト一覧から対象を選択する。
  2. 変更内容を入力して保存する。
  3. クローズ操作を行うと、以降の工数・経費登録が不可になる。

---

## UC-PJ03: 予算の設定

- **アクター**: 管理者
- **概要**: プロジェクトに対して予算（工数コスト・経費・合計）を設定する。
- **基本フロー**:
  1. 管理者が対象プロジェクトを選択する。
  2. 工数予算（時間または金額）・経費予算・売上目標を入力して保存する。
- **事後条件**: 設定した予算が原価・売上レポートの比較基準として使用される。

---

## UC-PJ04: 経費の登録

- **アクター**: 社員、管理者
- **概要**: プロジェクトに直接かかった経費（交通費・外注費・備品など）を登録する。
- **基本フロー**:
  1. 対象プロジェクトと経費カテゴリ（交通費 / 外注費 / 消耗品 / その他）を選択する。
  2. 金額・発生日・摘要を入力して保存する。
- **例外フロー**:
  - クローズ済みプロジェクトには経費を登録できない。

---

## UC-PJ05: 経費の編集・削除

- **アクター**: 社員、管理者
- **概要**: 登録済みの経費エントリを修正または削除する。
- **基本フロー**:
  1. 経費一覧から対象エントリを選択する。
  2. 修正して保存、または削除を確認して実行する。
- **例外フロー**:
  - 締め済み期間の経費は編集・削除不可とし、エラーを表示する。

---

## UC-PJ06: 原価の集計・確認

- **アクター**: 管理者
- **概要**: プロジェクトの原価（工数コスト＋経費）を集計して確認する。工数コストは社員の単価×工数時間で算出する。
- **基本フロー**:
  1. 管理者が対象プロジェクトと集計期間を指定する。
  2. システムが以下を算出して表示する:
     - 工数コスト（社員別の単価 × 工数時間の合計）
     - 経費合計（カテゴリ別内訳含む）
     - 原価合計（工数コスト＋経費）
  3. 設定済みの予算との差異（予算消化率）も併せて表示する。

---

## UC-PJ07: 売上の登録

- **アクター**: 管理者
- **概要**: プロジェクトに対する売上（請求額・入金額）を登録する。
- **基本フロー**:
  1. 管理者が対象プロジェクトを選択する。
  2. 売上金額・計上日・摘要（請求書番号など）を入力して保存する。
  3. 複数回に分けて登録可能（分割請求・マイルストーン請求に対応）。

---

## UC-PJ08: タスクカテゴリマスタの管理

- **アクター**: 管理者
- **概要**: 社員が工数登録時に選択するタスクカテゴリ（作業種別）を管理者が定義・管理する。社員は自由入力ではなくこのカテゴリから選択する。
- **基本フロー**:
  1. 管理者がカテゴリ名を入力して追加する（例: 開発 / 打ち合わせ / 調査 / レビュー / 管理・その他）。
  2. 既存カテゴリの名称を変更できる。
  3. 使用しなくなったカテゴリを無効化する（過去データへの影響はなし）。
- **例外フロー**:
  - 有効なカテゴリが0件になる操作は不可とする。

---

## UC-PJ09: 収支レポートの確認

- **アクター**: 管理者
- **概要**: プロジェクト単位で予算・原価・売上・利益を一覧で把握する。
- **基本フロー**:
  1. 管理者が期間またはプロジェクトを指定する。
  2. システムが以下を表示する:
     - 売上合計 / 売上目標比
     - 原価合計（工数コスト＋経費）/ 予算比
     - 粗利（売上 − 原価）/ 粗利率
     - タスクカテゴリ別の工数内訳
  3. CSV形式でダウンロードできる。
