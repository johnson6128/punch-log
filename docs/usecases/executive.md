# ユースケース: 経営ダッシュボード

経営層向けの閲覧専用レポート群。データの登録・変更は行わず、全社横断の集計結果を参照する。

---

## UC-EX01: 年度業績サマリーの確認

- **アクター**: 経営層
- **概要**: 今年度の売上・原価・粗利・原価率を、確定済み実績と見込みを合算して把握する。
- **基本フロー**:
  1. 経営層が対象年度を選択する（デフォルト: 当年度）。
  2. システムが以下を算出して表示する:

     | 区分 | 内容 |
     |---|---|
     | **確定売上** | 締め済み期間の計上売上合計 |
     | **実行中見込み** | 実行中プロジェクトの売上目標残額（目標 − 確定済み売上） |
     | **パイプライン** | 提案中・稟議中案件の売上目標合計（参考値） |
     | **売上合計（見込み含む）** | 確定売上 ＋ 実行中見込み |
     | **原価合計** | 確定工数コスト ＋ 確定経費 ＋ 実行中の発生済み原価 |
     | **粗利** | 売上合計（見込み含む） − 原価合計 |
     | **原価率** | 原価合計 ÷ 売上合計（見込み含む） |

  3. 月別の売上・原価・粗利の推移を一覧表示する。
  4. 前年度同期比を併せて表示する。
  5. CSV形式でダウンロードできる。
- **備考**: パイプラインは稟議承認前のため確度が低く、参考値として別区分で表示する。実行中見込みは営業が登録した案件売上目標をベースとする（UC-SL01 / UC-SL04）。

## UC-EX02: 案件パイプラインの確認

- **アクター**: 経営層
- **概要**: 全ステータスの営業案件を一覧し、ステータス別の売上見込み総額と案件の進捗を把握する。
- **基本フロー**:
  1. 経営層が対象年度・ステータスフィルターを指定する（複数選択可）。
  2. システムがステータス別の集計サマリーを表示する:
     - 提案中: 件数・売上目標合計
     - 稟議中: 件数・売上目標合計
     - 実行中: 件数・売上目標合計・確定売上合計・達成率
     - 完了: 件数・最終売上合計
  3. 案件一覧を表示する（案件No・プロジェクト名・担当営業・売上目標・ステータス・稟議No.・期間）。
  4. 案件を選択すると、当該プロジェクトの収支概況（UC-PJ09 の要約）を確認できる。
- **備考**: 受注確度の高いパイプラインを把握し、リソース計画や資金計画の判断材料として使用する。

## UC-EX03: キャッシュフローの確認

- **アクター**: 経営層
- **概要**: 月次の入金・支出の実績と見込みを確認し、資金繰りの状況を把握する。
- **基本フロー**:
  1. 経営層が対象期間を選択する。
  2. システムが月次のキャッシュフローを表示する:

     | 項目 | 内容 |
     |---|---|
     | **入金（確定）** | 締め済み期間の売上計上額 |
     | **入金（見込み）** | 実行中案件の売上目標残額を期間内に按分した予測値 |
     | **支出（確定）** | 締め済み期間の人件費（工数コスト＋割増）＋経費合計 |
     | **支出（見込み）** | 実行中プロジェクトの予算残額を残期間で均等按分した予測値（予算残額 ＝ 工数予算＋経費予算 − 発生済み支出） |
     | **月次収支** | 入金合計 − 支出合計 |
     | **累計収支** | 期間開始からの累計 |

  3. 月別の入金・支出・収支の推移を一覧表示する。
  4. CSV形式でダウンロードできる。
- **備考**: 入金・支出の見込みはプロジェクトの売上目標・予算残額を残期間で均等按分した概算値。支出の予算残額は「工数予算＋経費予算（コンティンジェンシー予備費は除く）− 発生済み工数コスト − 発生済み経費」で算出する。実際の入金タイミングは売上計上日（UC-PJ07 / UC-SL05）に依存する。

## UC-EX04: 全社稼働・人件費の確認

- **アクター**: 経営層
- **概要**: 全社員の稼働状況（プロジェクト別工数）と人件費を月次・年度単位で確認する。リソースの偏りや人件費の過大・過小を把握するために使用する。
- **基本フロー**:
  1. 経営層が対象期間・集計軸（社員別 / プロジェクト別 / 部門別）を指定する。
  2. システムが以下を算出して表示する:
     - 稼働工数（通常・残業・深夜・休日出勤の内訳）
     - 人件費合計（通常コスト＋割増コスト）
     - 稼働率（実績工数 ÷ 所定労働時間）
  3. 36協定アラートが発生している社員を別途ハイライト表示する（UC-WG08 参照）。
  4. CSV形式でダウンロードできる。
- **備考**: 個人の給与額は表示せず、コスト計算用の単価をもとに算出した工数コストのみを表示する。
