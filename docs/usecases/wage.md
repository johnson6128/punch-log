# ユースケース: 賃金・割増率管理

割増賃金の計算に必要なマスタと設定を管理する。
社員の雇用形態によって単価・割増率の管理元が異なる。

- **正社員・契約社員**: 職位マスタに基本時給と割増率を設定する。
- **派遣社員**: 派遣元会社マスタに時給と割増率を設定する。

---

## UC-WG01: 職位マスタの管理

- **アクター**: 管理者
- **概要**: 正社員・契約社員の職位ごとに基本時給と割増率を設定・管理する。
- **基本フロー**:
  1. 管理者が職位名（例: 一般職 / 主任 / 係長 / 管理職）を登録する。
  2. 各職位に以下を設定する:
     - 基本時給
     - 残業割増率（法定下限: 1.25、例: 1.30）
     - 深夜割増率（法定下限: 1.25、例: 1.25）
     - 休日出勤割増率（法定下限: 1.35、例: 1.35）
     - 深夜残業割増率（残業＋深夜の組み合わせ、法定下限: 1.50）
  3. 既存職位の名称・割増率を変更できる。
  4. 使用しなくなった職位を無効化する（過去データへの影響はなし）。
- **備考**: 管理職など労働基準法上の残業規定が適用されない職位は、割増率を0に設定できる。

---

## UC-WG02: 派遣元会社マスタの管理

- **アクター**: 管理者
- **概要**: 派遣社員の所属会社（派遣元）ごとに契約時給と割増率を設定・管理する。
- **基本フロー**:
  1. 管理者が派遣元会社名を登録する。
  2. 各派遣元会社に以下を設定する:
     - 基本時給（契約上の単価）
     - 残業割増率
     - 深夜割増率
     - 休日出勤割増率
     - 深夜残業割増率
  3. 既存会社の名称・割増率を変更できる。
- **備考**: 派遣元会社によって割増の計算方式が異なる場合があるため、会社単位で個別設定できる。

---

## UC-WG03: 割増判定ルールの設定

- **アクター**: 管理者
- **概要**: どの条件で各割増が適用されるかを定義する。
- **基本フロー**:
  1. 管理者が以下のルールを設定する:
     - **残業**: 1日の所定労働時間（例: 8時間）を超えた分
     - **深夜**: 深夜帯の時間範囲（例: 22:00〜5:00）
     - **休日出勤**: 法定休日（日曜）および会社指定の休日カレンダー
  2. 会社の就業規則に合わせてカスタマイズできる。
- **備考**: 残業＋深夜が重複する場合は、各割増率を合算して適用する。

---

## UC-WG04: 社員への単価・職位の紐付け

- **アクター**: 管理者
- **概要**: 各社員に職位（正社員）または派遣元会社（派遣社員）を紐付け、原価計算に使用する単価・割増率を確定する。
- **基本フロー**:
  1. 管理者が社員を選択する。
  2. 雇用形態（正社員 / 契約社員 / 派遣社員）を選択する。
  3. 正社員・契約社員の場合: 職位マスタから職位を選択する。
  4. 派遣社員の場合: 派遣元会社マスタから所属会社を選択する。
  5. 必要に応じて個人単位で時給を上書き設定できる（例: 同職位でも個別交渉がある場合）。
- **事後条件**: 設定した単価・割増率が工数コスト計算に反映される。

---

## UC-WG05: 割増賃金の確認

- **アクター**: 管理者
- **概要**: 指定期間の打刻データをもとに、社員ごとの残業・深夜・休日出勤時間と割増コストを確認する。
- **基本フロー**:
  1. 管理者が対象社員と期間を指定する。
  2. システムが打刻データと割増判定ルールをもとに以下を算出する:
     - 通常勤務時間・コスト
     - 残業時間・割増コスト
     - 深夜時間・割増コスト
     - 休日出勤時間・割増コスト
     - 合計人件費
  3. プロジェクト別の工数に割増コストを按分して原価へ反映する。
